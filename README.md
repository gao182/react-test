# 番茄闹钟线上webApp

## 实现功能

1. 简单用户系统---登陆/注册
2. 番茄闹钟---5分钟定时闹钟
3. 待办任务---番茄闹钟期间完成待办任务，自动记录到番茄描述中
4. 数据统计
5. 月统计数据---每月累计量 & 完成率 
6. 番茄历史统计---每日记录（可删改）
7. 待办任务统计---每日记录（可删改）

## 技术栈

- react、react-router、react-redux
- ant-d
- scss
- axios
